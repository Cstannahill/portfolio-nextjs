## 🧠 Overview

**Trivia App** is a full-stack, interactive trivia game designed for web and mobile platforms. It features a clean, responsive UI, real-time scoring, and a variety of question types. The app is built for scalability and engagement, making it ideal for both casual users and competitive play.
