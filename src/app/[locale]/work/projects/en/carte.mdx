## Overview

Carte is a multi-functional SaaS platform developed to streamline operations and provide actionable insights for organizations. The project included building an **analytics dashboard** for administrators to visualize critical business metrics and a **menu management system** to facilitate real-time updates of menu items and modifications. The platform's goal was to enhance efficiency and support data-driven decision-making while providing a user-friendly experience for both management and end-users.

## Key Features

- **Dynamic Analytics Dashboard**: Developed an interactive admin dashboard to display revenue trends, user activity, and order statistics, powered by complex SQL queries and data visualization tools.
- **Menu Management System**: Designed a flexible system allowing real-time updates to menu items and modifiers, ensuring accuracy and consistency across organizations.
- **User Roles and Permissions**: Implemented role-based access controls to ensure secure and tailored access for administrators, staff, and partners.
- **Database-Driven Architecture**: Designed and maintained a robust SQL Server database to handle large-scale operations efficiently.

- **.NET (C#)**: To create scalable and maintainable backend APIs.

## Challenges and Learnings

One of the primary challenges was managing the complexity of integrating real-time updates with database-driven operations, especially in a multi-organization environment. Designing efficient SQL queries and stored procedures for data analytics required careful optimization to handle large datasets while maintaining performance. Additionally, aligning the menu management system with the diverse needs of partner organizations highlighted the importance of creating modular and scalable solutions.

## Outcome

Carte successfully delivered a platform that enabled organizations to monitor and manage their operations effectively. The **analytics dashboard** provided actionable insights that improved decision-making, while the **menu management system** reduced overhead and ensured real-time accuracy. The project demonstrated the value of combining modern web technologies with thoughtful design to create impactful solutions.
