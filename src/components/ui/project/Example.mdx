## 🧩 Challenges & Solutions

<ProjectChallengeCard
  title="Performance Optimization"
  challenge="Initial component renders were slow on mobile devices, affecting user experience with noticeable lag during interactions."
  solution="Implemented React.memo for pure components, added virtualization for long lists, and optimized re-renders by refactoring state management with proper dependency arrays."
  impact="Reduced Time to Interactive by 45% and improved Lighthouse performance score from 74 to 92."
  difficulty="hard"
  domain="Performance"
/>

<ProjectChallengeCard
  title="Cross-Browser Compatibility"
  challenge="UI components displayed inconsistently across different browsers, particularly with CSS Grid and Flexbox layouts."
  solution="Created a comprehensive cross-browser testing strategy and implemented specific fallbacks for older browsers. Added a browser detection utility to serve optimized styles."
  impact="Achieved consistent appearance and functionality across Chrome, Firefox, Safari, and Edge, including support for IE11 with graceful degradation."
  difficulty="medium"
  domain="Frontend"
